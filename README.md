Beerrace - Zeitnahme und Auswertung eines Bierkastenrennen

Diese Webapp dient in erster Linie zum ausprobieren diverse Features.
Im Zielbetrieb soll sie die Zeitnahme und Dokumentation für ein Bierkastenrennen darstellen.

1. Alle Teilnehmenden Teams bekommen einen QR-Code an den Bierkasten.
2. An jeder Station wird der Code gescannt und die getrunken Flaschen sowie erreichte Punktzahl bei den Aktionen eingetragen.
3. Alle Daten werden anschließend in einer Datenbank mot Zeitstwmpel gespeichert.

Die Auswertung erfolgt im Ziel automatisiert:
1. Die geringste Gesamtzeit zwischen Start und Ziel
2. Strafminuten nicht getrunkener oder Gelehrter Flaschen
3. Punkte für die Zwischdnstationen
   
Es lässt sich damit auch eine aktuelle Punktetabelle erstellen und durch die Teilnehmer / im Ziel eine aktuelle Liste aller Teilnehmer anzeigen.
